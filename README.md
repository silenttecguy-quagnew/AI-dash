| Tab | Description |
|---|---|
| 🏠 **Home** | Daily Command Centre — MRR, ARR, quick execute, revenue milestones, weekly wins |

| 🎭 **Avatars** | Autonomous Avatar Builder + HeyGen/D-ID motion video generator + avatar folder |

| 🌟 **Dream Build** | Dream Build Enquiry — lead magnet for custom dashboard sales |
